# Analyzing the Influence of Pricing and Logistics Factors on Customer Satisfaction Using Machine Learning Techniques

## Project Overview

This project investigates how pricing and logistics factors influence customer satisfaction in e-commerce using the Olist Brazilian E-Commerce Dataset.

Machine learning techniques are applied to identify patterns affecting customer reviews and satisfaction.

## Dataset

- Olist Brazilian E-Commerce Dataset
- Source: Kaggle

## Machine Learning Techniques

- K-Means Clustering
- Logistic Regression

## Features Used

- Price
- Freight Value
- Payment Value
- Payment Installments
- Delivery Time
- Delivery Delay

## Target Variable

- Positive Review (Derived from Review Score)

## Evaluation

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
